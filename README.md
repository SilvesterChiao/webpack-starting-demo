## 脚本

-   npm run webpack：webpack 打包
-   npm run dev：development 模式打包
-   npm run server：webpack-dev-server 启动
-   npm run build：production 模式打包
-   npm run babel：编译 es6
-   npm run lint: 语法检查
-   npm run lint-fix: 语法检查并 fix
-   npm run prettier：使用 prettier 格式化
