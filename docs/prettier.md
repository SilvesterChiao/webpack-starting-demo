# prettier

## 安装

```bash
npm install -g prettier
npm install --save-dev prettier
```

## option

-   printWidth: 80,
-   tabWidth: 8,
-   useTabs: false,
-   singleQuote: 强制单引号,
-   semi: 行尾分号,
-   bracketSpacing: false,
-   arrowParens: "always",
-   trailingComma: "all"
